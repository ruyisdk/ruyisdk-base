# 包管理器
"用户在PC端可使用基础包管理器 管理 duo上的包/程序（安装/禁用/卸载）（安装路径）

1. 用户在PC端可使用在线包管理 快速安装 【官方】【非官方】包*
2. 支持开发者 上传/维护 自己开发的包*
3. 对于包：要求PC端和Duo能同步，即满足交叉编译条件
4. 对于程序：要求PC端能控制Duo端 安装/卸载"

# 设备终端访问器

1. 支持自动搜索设备
2. 支持一键配置RNDIS访问Duo
3. 支持访问串口（波特率自定义）

# 交叉编译

1. 交叉编译
2. 编译后 在debug文件夹生成可执行文件
3. 编译后自动上传至Duo
4. 编译后自动上传至Duo并运行"

# 快速新建项目

1. 一键生成空项目（带目录结构）
2. 带大小核通信的main.c （hello world示例）
3. 支持在一个项目里完成对两个核的程序开发及核间通信开发"

# 快速创建示例项目
1. 包含【blink】【pwm】【I2c】【spi】【uart】【adc】【wifi】【bt】【mic】……等基础项目，需包含大小核通信示范。
2. 包含至少一个【服务（大核）+传感器与外设（小核）】项目，包含大小核通信示范

# 设备资源管理器
1. 在PC端监控（cpu0 / cpu1占用、内存占用、网络占用）
2. 在PC端查看进程
3. 在PC端控制进程（Kill）

# 开机自启动项管理
1. 在PC端管理Duo自启动项（添加/删除）

# 大小核内存分配管理
1. 在PC端控制Duo 大小核内存分配
2. 提示重启生效
